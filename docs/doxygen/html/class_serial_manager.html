◆ sendCommand()

void SerialManager::sendCommand ( uint8_t cmdId,
const QByteArray & data = {} )

Envia un comando al MCU encapsulado en una trama UNER.

Construye y envia una trama UNER completa al MCU.

Parámetros
cmdIdIdentificador del comando (ver Protocol.h).
dataDatos adicionales del comando (puede estar vacio).

Formato de trama:

[ 'U' | 'N' | 'E' | 'R' | LEN | ':' | CMDID | DATA... | CKS ]

LEN = 1 (CMDID) + data.size() + 1 (CKS). CKS = XOR de todos los bytes desde 'U' hasta el ultimo dato.

Parámetros
cmdIdIdentificador del comando a enviar.
dataBytes de datos adicionales (puede estar vacio).

Envia un comando CMD_ALIVE al MCU como heartbeat periodico.

Se ejecuta cada 500 ms mediante el QTimer aliveTimer. El MCU responde con EVT_ALIVE para confirmar que esta operativo.

◆ processRxByte()

void SerialManager::processRxByte ( uint8_t byte)
private

Procesa un byte recibido a traves de la FSM de decodificacion UNER.

Parámetros
byteByte recibido del puerto serie.

La FSM sincroniza con el encabezado "UNER", captura LEN, el separador ':', y finalmente el payload byte a byte hasta verificar el checksum. Al completar una trama valida, emite frameReceived().

Parámetros
byteByte recibido del puerto serie.
