Descripción detallada

Ventana principal del HMI de la cinta clasificadora.

Hereda de QMainWindow. Gestiona toda la interfaz grafica, la comunicacion con el MCU via SerialManager, y el estado local de la aplicacion (conteo de cajas, cola de transito, etc.).
